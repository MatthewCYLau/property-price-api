# Property Price API

An ASP.NET Core web API which returns suggested property offer price

## Pre-requisite

Ensure you have installed [.NET SDK](https://dotnet.microsoft.com/en-us/download):

```bash
dotnet --version
```

## Usage

```bash
dotnet run
```
